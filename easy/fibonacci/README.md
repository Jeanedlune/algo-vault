# Fibonacci Sequence

**Difficulty:** Easy

## Problem

Generate Fibonacci sequence up to `n` terms.

### Example

Input: `7`  
Output: `[0, 1, 1, 2, 3, 5, 8]`

### Constraints

- `0 ≤ n ≤ 10^3`
- Return an empty list if n = 0.

### Test Cases

| Input | Expected Output |
| :---- | :-------------- |
| 0     | []              |
| 1     | [0]             |
| 2     | [0,1]           |
| 5     | [0,1,1,2,3]     |
| 7     | [0,1,1,2,3,5,8] |

### Complexity

- Time: O(n)
- Space: O(n)
