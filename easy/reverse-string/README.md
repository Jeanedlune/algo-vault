# Reverse String

## Problem Description

Given a string, return its reverse.

### Example

**Input:** `"hello"`  
**Output:** `"olleh"`

### Constraints

- The input can contain letters, numbers, spaces, or symbols.
- Handle empty strings correctly.

### Test Cases

| Input           | Expected Output |
| --------------- | --------------- |
| "hello"         | "olleh"         |
| "Hacktoberfest" | "tsefrebotkcaH" |
| "Python"        | "nohtyP"        |
| "a"             | "a"             |
| ""              | ""              |

### Complexity

- **Time Complexity:** O(n)
- **Space Complexity:** O(n)
